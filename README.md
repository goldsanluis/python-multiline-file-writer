# Multiline File Writer

A Python program that asks the user to enter lines of text one at a time
and writes them all into a file called mylife.txt. The program keeps
asking for more lines until the user types n.

## What It Does
- Prompts the user to enter a line of text
- Asks if there are more lines to add
- Repeats until the user answers n
- Saves all entered lines into mylife.txt

## Sample Interaction
Enter line: Beautiful is better than ugly.
Are there more lines y/n? y
Enter line: Explicit is better than implicit.
Are there more lines y/n? y
Enter line: Simple is better than complex.
Are there more lines y/n? n

## Output File
- mylife.txt — contains all the lines entered by the user

## Requirements
- Python 3.x
